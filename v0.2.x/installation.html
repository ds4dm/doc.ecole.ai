Runtime dependencies¶

  • Scip >= 6.0 needs to be independently installed with CMake. Configure with -D PARASCIP=true and make an installation. For instance, in scipoptsuite root directory, this could look like

    cmake -B build/ -D PARASCIP=true
cmake --build build/ --parallel
cmake --install build/

Building¶

Depending on what you are building (the C++ library, the Python bindings, the tests, the documentation) a number of tools are required. A superset of these tools are listed in the Conda environment file conda-dev.yml.

All components can be build with CMake. For instance

cmake -B build/ -D CMAKE_BUILD_TYPE=release
cmake --build build/ --parallel

The python package can then be installed from the build directory

pip install build/python/
