Listing¶

Is Done¶

class ecole.reward.IsDone¶

Single reward on terminal states.

obtain_reward(self: ecole.reward.IsDone, state: ecole::environment::State, done: bool = False) → float¶

Return 1 if the episode is on a terminal state, 0 otherwise.

reset(self: ecole.reward.IsDone, state: ecole::environment::State) → None¶

Do nothing.

LP Iterations¶

class ecole.reward.LpIterations¶

LP Iteration difference.

The reward is defined as the number of iterations done solving the Linear Program associated with the problem since the previous state.

obtain_reward(self: ecole.reward.LpIterations, state: ecole::environment::State, done: bool = False) → float¶

Update the internal LP iteration count and return the difference.

The difference in LP iteration is computed in between calls.

reset(self: ecole.reward.LpIterations, state: ecole::environment::State) → None¶

Reset the internal LP iterations count.

Arithmetic¶

class ecole.reward.Arithmetic¶

Proxy class for doing arithmetic on reward functions.

An object of this class is returned by reward functions operators to forward calls to the reward functions parameters of the operator.

obtain_reward(self: ecole.reward.Arithmetic, state: object, done: bool = False) → float¶

Obtain the reward of result of the operator.

Call obtain_reward on all reward functions parameters that were used to create this object and compute the operation on the results.

reset(self: ecole.reward.Arithmetic, state: object) → None¶

Reset the reward functions of the operator.

Call reset on all reward functions parameters that were used to create this object.
